## 📚 Idea/Concepto 

Es una red virtual aislada lógicamente dentro de la nube pública, implementada como Network as a Service (NaaS). Se puede crear una red privada, controlar las direcciones IP y manejar reglas de seguridad. Todo esto se configura mediante APIs, CLI o UI, sin necesidad de gestionar hardware físico. Además, permite conectividad híbrida mediante gateways y túneles VPN para enlazar con centros de datos on-premise o internet. Gracias al SDN (Software Defined Networking) , se puede aprovisionar, escalar o destruir infraestructuras de red complejas en minutos.
