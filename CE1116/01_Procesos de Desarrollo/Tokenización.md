## 📚 Idea/Concepto 

La tokenización es el proceso en el cual se separa en partes más pequeñas un texto. Estas partes se llaman tokens, que funcionan como la unidad mínima de lenguaje que el modelo utiliza para trabajar.
Se divide el texto para tener partes más fácilmente identificables. Esto se podría hacer palabra por palabra, pero también se podría, por ejemplo, separar una palabra entre el concepto de la palabra y su terminación.
Además, cada una de estas partes se asocia a un identificador numérico dentro de un vocabulario, formando una secuencia de Token IDs que el modelo puede manejar como tensores. Este proceso también permite la decodificación, es decir, convertir nuevamente esos identificadores en texto legible.
