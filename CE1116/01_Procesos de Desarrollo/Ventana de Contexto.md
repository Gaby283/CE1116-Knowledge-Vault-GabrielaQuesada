## 📚 Idea/Concepto 

Es el rango de información que una IA puede procesar al mismo tiempo, similar a una memoria de trabajo. Esta capacidad se mide en tokens, que son las unidades que el modelo utiliza para procesar el texto.
La ventana de contexto define el límite máximo de tokens sobre los cuales el mecanismo de autoatención puede calcular dependencias y relaciones al mismo tiempo.
Si se tiene una interacción muy larga con una IA, puede llegar un punto en el que el modelo ya no pueda considerar la información más antigua dentro de esa ventana. Por ejemplo, si la ventana permite procesar una cantidad limitada de tokens y la interacción supera ese límite, la IA ya no podrá procesar la información que quedó fuera de ese rango. Además, en esta ventana no solo se incluye la conversación, sino también prompts de sistema, documentos adjuntos o datos recuperados (RAG), lo que puede hacer que este recurso se agote antes.
