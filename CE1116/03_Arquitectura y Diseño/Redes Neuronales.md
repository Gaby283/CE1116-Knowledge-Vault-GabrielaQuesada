## 📚 Idea/Concepto 

Las IAs utilizan redes neuronales artificiales formadas por capas interconectadas de neuronas artificiales, que en realidad son funciones matemáticas.
Cada nodo recibe información de otros nodos y la transforma utilizando pesos, sesgos y una función de activación, y luego transmite el resultado a otros nodos. El objetivo de estas conexiones es que la red pueda aprender patrones a partir de los datos de entrada y así generar una salida.
El aprendizaje ocurre cuando la red ajusta estos parámetros para minimizar una función de costo, utilizando un proceso llamado retropropagación, que permite modificar los pesos según el error cometido.
