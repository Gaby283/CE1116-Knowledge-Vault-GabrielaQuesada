## 📚 Idea/Concepto 

Es la combinación de Cloud Público y Cloud Privado conectados entre sí, permitiendo que aplicaciones y cargas de trabajo se muevan entre ambos entornos mediante una arquitectura de interoperabilidad que garantiza la portabilidad de los activos. Se usa un plano de control unificado para gestionar los recursos de manera consistente. Esto permite que una empresa ejecute cargas sensibles en la nube privada y use la nube pública para absorber picos de demanda cuando la capacidad privada se satura.
